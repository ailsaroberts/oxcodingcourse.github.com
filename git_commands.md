# Git Commands

## Standard usage

    git status
    git add myfile.html
    git add .
    git commit -m "Provide a description of the changes here" 
    git log
    gitk --all

## Sharing with others

    git remote add <some github repo>
    git push
    git fetch
    git branch -a
    git merge origin/master
